Olvídate de instalar drivers o software de escáneres. DocScan se comunica directamente con escáneres de red compatibles con eSCL/AirScan vía HTTP. También puedes usar la cámara de cualquier dispositivo para capturar documentos al vuelo, o simplemente arrastrar archivos PDF e imágenes al dashboard.
